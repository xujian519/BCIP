# BCIP Agent 测试版说明

## 🎯 测试范围

本次测试面向 **10 人以内** 的内部测试团队。

## 📦 安装说明

### macOS

由于这是**未签名版本**，安装时需要额外步骤：

#### 方法 1：通过 DMG 安装（推荐）

1. 下载 `BCIP-Agent_0.1.0_x64.dmg`
2. 双击打开 DMG 文件
3. 将 **BCIP Agent** 拖到 **Applications** 文件夹
4. 首次启动时，系统会提示"无法验证开发者"：
   - 点击 **取消**
   - 打开 **系统偏好设置** → **安全性与隐私**
   - 点击下方的 **仍要打开**
   - 或者使用快捷键：按住 `Control` 键点击应用图标，选择 **打开**

#### 方法 2：通过 APP 直接运行

1. 下载 `BCIP Agent.app`
2. 将应用拖到桌面或应用程序文件夹
3. 首次启动时：
   - 按住 `Control` 键点击应用图标
   - 选择 **打开**
   - 在弹出的对话框中点击 **打开**

#### 如果仍然无法打开

打开终端，运行以下命令清除隔离属性：

```bash
xattr -cr "/Applications/BCIP Agent.app"
```

然后再次尝试启动。

## 🔧 环境要求

- **macOS**: 11.0 (Big Sur) 或更高版本
- **内存**: 建议 8GB 以上
- **存储**: 至少 500MB 可用空间

## 🚀 快速开始

1. 启动 **BCIP Agent**
2. 在登录界面选择 **本地模式**（无需登录）
3. 进入主界面后：
   - **左侧**: 文件浏览器
   - **中间**: 工作区（权利要求、对比、审查等）
   - **右侧**: 智能对话面板
4. 首次启动会自动检测 `bcip` 并连接 app-server（stdio / unix proxy，无需 WebSocket 端口）
5. 已连接时中心区为 Agent 工作区；未连接或浏览器 `npm run dev` 时可能显示专利演示视图（带「演示」横幅）

## 🖼 macOS Dock 图标

Dock 应显示**圆角矩形（squircle）**，而不是正矩形。若仍为正方形：

1. 确认已用**最新打包**的 `.app` / DMG（旧构建会缓存错误的 `icon.icns`）。
2. 开发者更新图标后执行：
   ```bash
   cd apps/desktop && npm run icons:generate
   npm run tauri:build:bundle
   ```
3. 源图为 `src-tauri/icons/logo.png`；脚本会去除黑边/白底、铺满 macOS 渐变背景，并生成 `icon.icns` / `icon.ico` 及 Tauri 所需的全部 PNG。
4. `tauri dev` 与正式包可能图标不一致；以 **打包后的 `.app`** 为准验收 Dock。

## ⚠️ 已知限制

1. **未签名**: 首次启动需要手动允许
2. **自动更新**: 测试版不支持自动更新
3. **连接**: 桌面端会尝试 attach/spawn app-server；失败时可点 Agent 底栏「重试」
4. **性能**: 未经优化，可能存在内存占用较高的情况
