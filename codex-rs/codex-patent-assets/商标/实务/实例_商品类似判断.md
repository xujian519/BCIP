## 📋 商品类似判断标准

### 核心判断因素

根据《商标审查审理指南》和司法解释，商品类似判断需考虑以下因素：

| 判断因素 | 内容 | 判断要点 |
|----------|------|----------|
| **功能用途** | 商品的实际用途 | 用途是否相同或关联 |
| **生产部门** | 制造商是否相同 | 生产工艺是否相似 |
| **销售渠道** | 销售途径是否相同 | 销售场所是否重叠 |
| **消费对象** | 目标消费群体 | 消费群体是否相同 |
| **原材料** | 生产原材料 | 原材料是否相同 |
| **服务内容** | 服务目的 | 服务内容是否相同 |

### 类似商品定义

> 类似商品是指在功能、用途、生产部门、销售渠道、消费对象等方面**相同**，或者相关公众一般认为其存在**特定联系**、**容易造成混淆**的商品。

---

## 📊 类似判断原则

### 原则一：功能用途相同

```
示例：
- 电视机 vs 显示器 → 类似（视频显示功能）
- 服装 vs 鞋子 → 不类似（功能不同）
- 手机 vs 电脑 → 类似（通讯/计算设备）
```

### 原则二：特定联系

```
"特定联系"的判断：
- 商品之间存在上下游关系
- 商品之间经常同时使用
- 商品属于同一系统或配套使用
- 商品在销售渠道、消费群体上高度重合

示例：
- 打印机 vs 墨盒 → 类似（配套使用）
- 电脑 vs 鼠标 → 类似（配套使用）
- 服装 vs 纽扣 → 类似（原材料关联）
```

### 原则三：相关公众认知

```
"相关公众"的判断：
- 以普通消费者的认知为标准
- 考虑商品的实际销售场景
- 考虑行业惯例和消费习惯

示例：
- 眼镜店销售的眼镜 vs 眼科医院的眼镜 → 类似（消费者认知相同）
- 超市销售的饮料 vs 餐厅销售的饮料 → 类似（消费场景重叠）
```

## 🔍 类似判断示例

### 示例1：服装与配饰

| 商品A | 商品B | 判断 | 理由 |
|------|-------|------|------|
| 服装 | 帽子 | 类似 | 同属穿戴用品，消费群体相同 |
| 服装 | 鞋子 | 不类似 | 功能不同（覆盖 vs 行走） |
| 服装 | 腰带 | 类似 | 服装配搭使用 |
| 服装 | 袜子 | 类似 | 同属穿着搭配 |

### 示例2：电子产品

| 商品A | 商品B | 判断 | 理由 |
|------|-------|------|------|
| 手机 | 平板电脑 | 类似 | 功能重叠、渠道相同 |
| 手机 | 手机壳 | 类似 | 配套使用 |
| 手机 | 电脑 | 类似 | 同属数码产品 |
| 手机 | 相机 | 不类似 | 专业程度不同 |

### 示例3：食品饮料

| 商品A | 商品B | 判断 | 理由 |
|------|-------|------|------|
| 饮料 | 矿泉水 | 类似 | 同属饮品 |
| 饮料 | 牛奶 | 类似 | 均可饮用 |
| 饮料 | 糖果 | 不类似 | 食用方式不同 |
| 饮料 | 饼干 | 不类似 | 食用场景不同 |

## 📈 类似判断与商标近似的交互

### 核心规则

| 情形 | 商标近似 | 商标不近似 |
|------|----------|------------|
| **商品类似** | 侵权可能性高 | 侵权可能性低 |
| **商品不类似** | 需考虑驰 名商标 | 不构成侵权 |

### 驰 名商标跨类保护

> 就不相同或者不相类似商品申请注册的商标是复制、摹仿或者翻译他人已经在中国注册的驰 名商标，**误导公众**，致使该驰 名商标注册人的利益可能受到损害的，**不予注册并禁止使用**。

**要件**：
1. 引证商标为已注册驰 名商标
2. 诉争商标构成复制、摹仿、翻译
3. 误导公众
4. 致使驰 名商标注册人利益可能受损

## 📚 审查依据

### 法律规定

- **《商标法》第30条**：申请注册的商标不得与他人在同一种商品或类似商品上已注册商标近似
- **《商标法》第13条第3款**：驰 名商标跨类保护
