# 最高人民法院关于审理涉及驰 名商标保护的民事纠纷案件应用法律若干问题的解释

## 条款详解

### 第1条　案件类型与受理条件

> 为正确审理涉及驰 名商标保护的民事纠纷案件，根据《中华人民共和国民法典》《中华人民共和国商标法》《中华人民共和国民事诉讼法》等有关法律规定，结合民事审判实践，制定本解释。

**案件类型**：

| 类型 | 说明 |
|------|------|
| 侵权纠纷 | 未经许可在相同/类似商品上使用与驰 名商标相同/近似商标 |
| 合同纠纷 | 商标许可使用合同纠纷 |
| 不正当竞争 | 将驰 名商标作为字号使用等 |

> [!note] 受理条件
> 原告应当提供证据证明：
> 1. 其商标为驰 名商标
> 2. 被诉侵权信息为驰 名商标
> 3. 被诉侵权行为发生在驰 名商标认定之后

---

### 第2条　驰 名商标的认定

> 在下列民事纠纷案件中，当事人以商标法第十三条规定为依据主张权利的，人民法院根据当事人的请求和案件的具体情况，可以对涉及的注册商标是否驰 名依法作出认定：

**适用情形**：
（一）以驰 名商标复制、摹仿、翻译他人未在中国注册的驰 名商标为由提起的侵权诉讼；
（二）以驰 名商标复制、摹仿、翻译他人已在中国注册的驰 名商标为由提起的侵权诉讼；
（三）以企业名称与驰 名商标相同或者近似为由提起的侵权诉讼。

> [!tip] 认定原则
> 认定驰 名商标，应当依照商标法第十四条的规定进行。人民法院在审理商标纠纷案件中，根据当事人的请求和案件的具体情况，可以对涉及的注册商标是否驰 名依法作出认定。

**认定因素**（商标法第十四条）：
- 相关公众对该商标的知晓程度
- 该商标使用的持续时间
- 该商标的任何宣传工作的持续时间、程度和地理范围
- 该商标作为驰 名商标受保护的记录
- 该商标驰名的其他因素

---

### 第3条　驰 名商标的跨类保护

> 在不相同或者不相类似的商品或者服务上擅自使用他人的驰 名商标，在相关公众中造成混淆，使他人合法权益受到损害的，人民法院应当根据当事人的请求和案件的具体情况，认定这种行为是否构成商标侵权或者不正当竞争。

**保护范围**：
- **未注册驰 名商标**：在相同/类似商品上禁用
- **已注册驰 名商标**：在不相同/不类似商品上也可禁用（需满足"误导公众"要件）

> [!warning] 跨类保护要件
> 对于已注册驰 名商标的跨类保护，需要证明：
> 1. 商标构成驰 名商标
> 2. 被诉侵权行为误导公众
> 3. 致使驰 名商标注册人的利益可能受到损害

---

### 第4条　复制、摹仿、翻译的认定

> 下列行为属于商标法第十三条规定的"复制、摹仿或者翻译"他人驰 名商标的行为：

**具体形式**：

| 类型 | 含义 | 示例 |
|------|------|------|
| 复制 | 完全照搬 | 将驰 名商标图案直接使用 |
| 摹仿 | 实质性相似 | 改变部分但整体相似 |
| 翻译 | 语言转换 | 将外文商标译为中文 |

> [!important] 主要部分
> 对他人驰 名商标主要部分的复制、摹仿、翻译，也属于"复制、摹仿、翻译"。

**认定标准**：
- 整体外观相似
- 呼叫方式相似
- 含义相似
- 相关公众容易混淆

---

### 第5条　误导公众的认定

> 被诉侵权的商标构成驰 名商标，但被诉侵权行为不是在前述第（四）项、第（五）项、第（六）项所述的驰 名商标核定使用的商品或者服务上使用的，人民法院可以根据案件的具体情况，认定被诉侵权行为误导公众。

**判断因素**：

| 因素 | 说明 |
|------|------|
| 引证商标的显著性和知名度 | 驰 名程度越高，保护范围越宽 |
| 商标标识的近似程度 | 越近似越容易误导 |
| 商品/服务的关联程度 | 关联度越高，混淆可能性越大 |
| 相关公众的重合程度 | 消费群体重叠程度 |
| 其他因素 | 实际混淆证据、主观意图等 |

> [!tip] 实务要点
> - 需要提供证据证明驰 名商标的知名度
> - 需要证明被诉侵权行为误导了相关公众
> - 可以提供实际混淆的证据（如消费者投诉、市场调查报告等）

---

### 第6条　赔偿责任主体

> 原告以被告违反商标法第十三条的规定为由提起诉讼的，应当提供以下证据：

**举证要求**：

| 证据类型 | 内容 |
|----------|------|
| 主体证明 | 证明原告为驰 名商标注册人 |
| 驰 名证明 | 证明该商标为驰 名商标 |
| 侵权证明 | 证明被告实施了侵权行为 |
| 损害证明 | 证明原告权益受到损害 |

> [!note] 共同侵权
> 被告虽然没有直接实施侵权行为，但为他人实施侵权行为提供便利条件、帮助他人实施侵权行为的，人民法院可以认定为共同侵权。

---

### 第7条　赔偿数额的计算

> 人民法院在确定赔偿数额时，应当根据侵权行为的性质、期间、后果，侵权人的主观过错程度，商标的声誉及制止侵权行为的合理开支等因素综合确定。

**计算方式**（按优先顺序）：

| 方式 | 计算方法 | 适用条件 |
|------|----------|----------|
| 实际损失 | 销售减少量 × 单位利润 | 能证明损失 |
| 侵权获利 | 销售量 × 单位利润 | 损失难以确定 |
| | 侵权商品销售量 × 商品单位利润 | 单位利润无法查明时参照注册商标商品 |
| 许可费倍数 | 许可使用费 × 合理倍数 | 前两项难以确定 |
| 法定赔偿 | 500万元以下 | 前三项均难以确定 |

> [!important] 惩罚性赔偿
> 对恶意侵犯商标专用权，情节严重的，可以在按照上述方法确定数额的**一倍以上五倍以下**确定赔偿数额。

**证据规则**：
- 权利人已经尽力举证，而与侵权行为相关的账簿、资料主要由侵权人掌握的，可以责令侵权人提供
- 侵权人不提供或提供虚假的，可以参考权利人的主张和证据判定

---

### 第8条　制止侵权的合理开支

> 制止侵权行为所支付的合理开支，包括权利人或者委托代理人对侵权行为进行调查、取证的合理费用。

**合理开支范围**：

| 项目 | 说明 |
|------|------|
| 调查费 | 调查取证费用 |
| 取证费 | 证据保全费用 |
| 律师费 | 符合国家有关部门规定的律师费用 |
| 差旅费 | 诉讼相关差旅费用 |
| 公证费 | 证据公证费用 |

> [!tip] 律师费
> 人民法院根据当事人的诉讼请求和案件具体情况，可以将符合国家有关部门规定的律师费用计算在赔偿范围内。

---

### 第9条　证据保全与诉讼禁令

> 在诉讼中，人民法院可以根据当事人的请求和案件的具体情况，责令提供担保。

**证据保全**：
- 证据可能灭失或者以后难以取得的情况下，可以申请证据保全
- 人民法院可以主动采取证据保全措施

**诉讼禁令**：
- 权利人有证据证明他人正在实施或者即将实施侵权行为
- 如不及时制止将会使其合法权益受到难以弥补的损害的
- 可以依法在起诉前向人民法院申请采取责令停止有关行为和财产保全的措施

> [!warning] 担保要求
> 申请行为保全或者证据保全的，应当提供担保。不提供担保的，驳回申请。

---

### 第10条　法律适用

> 本解释未作规定的，适用《中华人民共和国商标法》《中华人民共和国反不正当竞争法》和《中华人民共和国民事诉讼法》以及其他法律和司法解释的规定。

**法律适用顺序**：
1. 本解释
2. 商标法
3. 反不正当竞争法
4. 民事诉讼法
5. 其他法律和司法解释

**竞合处理**：
- 驰 名商标与企业名称冲突：可以同时适用商标法和反不正当竞争法
- 侵权与不正当竞争竞合：当事人可选择一种诉由

## 📌 典型案例类型

### 案例类型一：跨类抢注

**案情**：A公司将其驰 名商标"B"注册在第25类（服装）上，B公司将该商标注册在第18类（皮革制品）上。

**裁判要点**：
- A公司商标为驰 名商标
- B公司在不相同类别上的注册误导公众
- 构成商标侵权

### 案例类型二：字号冲突

**案情**：A公司"华为"为通讯设备领域驰 名商标，B公司在第41类教育培训类将"华为"注册为企业字号。

**裁判要点**：
- 驰 名商标受反不正当竞争法保护
- 将驰 名商标作为企业字号使用误导公众
- 构成不正当竞争

### 案例类型三：域名抢注

**案情**：A公司"茅台"为酒类驰 名商标，B公司注册域名"maotai.com.cn"进行白酒销售。

**裁判要点**：
- 域名与驰 名商标近似
- 用于相同/类似商品交易
- 误导公众
- 构成商标侵权

## 📊 实务要点汇总

### 原告举证要点

| 要点 | 说明 |
|------|------|
| 驰 名证明 | 提供驰 名商标认定记录、使用时间、宣传范围等证据 |
| 侵权证明 | 证明被告使用了相同/近似标识 |
| 混淆证明 | 提供实际混淆证据（消费者投诉、市场调查报告等） |
| 损害证明 | 证明因侵权行为受到的实际损失 |

### 被告抗辩要点

| 抗辩类型 | 要件 |
|----------|------|
| 不驰 名 | 举证证明原告商标不构成驰 名商标 |
| 不近似 | 举证证明标识整体不近似 |
| 不混淆 | 举证证明不会误导相关公众 |
| 正当使用 | 证明使用行为属于正当使用（如描述性使用） |
| 已获许可 | 证明已获得原告授权 |

### 程序要点

| 要点 | 内容 |
|------|------|
| 管辖 | 侵权行为实施地、被告住所地法院 |
| 诉讼时效 | 3年，自知道或应当知道权利受到损害之日起计算 |
| 级别管辖 | 至少为中级人民法院 |
| 证据保全 | 可以在起诉前申请 |
