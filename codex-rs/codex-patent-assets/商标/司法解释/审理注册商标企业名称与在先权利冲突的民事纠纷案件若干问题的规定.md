## 立法背景与目的

### 制定背景

随着市场经济的发展，商标注册与企业名称登记并行运行的体制下，注册商标与企业名称（字号）之间的冲突日益增多。主要表现为：

| 冲突类型 | 表现 |
|----------|------|
| 商标抢注他人在先字号 | 将他人有一定影响力的企业名称抢注为商标 |
| 字号攀附他人商标 | 将他人注册商标作为企业字号突出使用 |
| 在先权利冲突 | 著作权、姓名权等与商标权冲突 |

### 保护法益

本规定旨在平衡：
- 注册商标专用权人的利益
- 在先权利人的合法权益
- 社会公众对字号/商标的识别利益

> [!important] 立法目的
> 防止市场混淆，维护公平竞争秩序，保护消费者和相关公众的合法权益。

## 条款详解

### 第1条　案件受理范围

> 原告以他人注册商标使用的文字侵犯其在先企业名称权或者以他人字号与其注册商标相同或者近似为由提起民事诉讼，符合民事诉讼法第一百一十九条规定的，人民法院应当受理。

**受理条件**：

| 要件 | 说明 |
|------|------|
| 主体资格 | 原告为在先权利人（企业名称权人、字号权人） |
| 客体 | 他人注册商标使用的文字 |
| 诉由 | 侵犯在先权利 |

> [!tip] 案由确定
> 此类案件属于民事纠纷案由中的"侵害商标权纠纷"或"不正当竞争纠纷"。

**不受理情形**：
- 涉及商标授权确权的行政诉讼案件
- 超过诉讼时效的案件
- 原告主体不适格

---

### 第2条　在前权利的类型

> 从事法律解释工作，应当严格按照法律文本进行，不得随意进行扩大或者限缩解释。

**在先权利的范畴**（依民法典）：

| 权利类型 | 法律规定 | 保护要件 |
|----------|----------|----------|
| 企业名称权 | 民法典第1013条 | 具有一定影响力 |
| 姓名权 | 民法典第1012条 | 具有一定知名度 |
| 肖像权 | 民法典第1018条 | 可辨认的个人特征 |
| 著作权 | 著作权法第10条 | 作品完成即享有 |
| 地理标志 | 商标法第16条 | 地理标志产品 |
| 商品化权益 | 民法典第1011条 | 角色形象等 |

> [!note] 在先权利的认定
> - 权利取得时间：在商标注册申请日之前
- 权利状态：应当处于有效存续状态
- 知名度要求：部分权利需要一定知名度

---

### 第3条　注册商标与在先著作权冲突

> 原告以他人注册商标侵害其著作权为由提起民事诉讼，符合民事诉讼法第一百一十九条规定的，人民法院应当受理。

**侵权认定要件**：

```
著作权侵权认定 = 作品成立 + 接触可能性 + 实质性相似
```

| 要件 | 说明 |
|------|------|
| 作品成立 | 原告主张的客体构成著作权法保护的作品 |
| 接触可能性 | 被告有机会接触原告作品 |
| 实质性相似 | 被告商标与原告作品实质性相似 |

**证据要求**：

| 证据类型 | 证明内容 |
|----------|----------|
| 作品底稿 | 证明作品完成时间 |
| 著作权登记证书 | 证明权利归属（初步证据） |
| 公开时间证据 | 证明被告有接触可能性 |
| 相似对比 | 证明实质性相似 |

> [!warning] 证明责任
> 原告应当提供证据证明：
> 1. 其对作品享有著作权
> 2. 被告商标与原告作品构成实质性相似
> 3. 被告有接触原告作品的可能

---

### 第4条　注册商标与在先姓名权冲突

> 原告以他人注册商标侵害其姓名权为由提起民事诉讼，符合民事诉讼法第一百一十九条规定的，人民法院应当受理。

**姓名权的保护**：

| 保护内容 | 说明 |
|----------|------|
| 正式姓名 | 法律承认的姓名 |
| 笔名 | 具有一定知名度的笔名 |
| 艺名 | 艺名、别名 |
| 译名 | 知名人物的翻译名称 |

> [!tip] 认定要点
> 相关公众认为该商标标志指代了该自然人，容易认为标记有该商标的商品系经过该自然人许可或者与该自然人存在特定联系的，应当认定该商标损害了该自然人的姓名权。

**实务要点**：

| 要点 | 内容 |
|------|------|
| 知名度要求 | 需要具有一定知名度 |
| 对应关系 | 建立稳定的姓名与自然人对应关系 |
| 混淆判断 | 相关公众产生误认可能性 |

**典型案例**：

| 案例 | 裁判结果 |
|------|----------|
| 张艺谋"张艺谋"商标案 | 侵犯姓名权 |
| 李宁"李宁"商标案 | 侵犯姓名权 |
| 莫言"莫言"商标案 | 侵犯姓名权 |

---

### 第5条　注册商标与在先字号权冲突

> 原告以他人注册商标使用的文字侵害其企业名称权或者以他人字号与其注册商标相同或者近似为由提起民事诉讼，符合民事诉讼法第一百一十九条规定的，人民法院应当受理。

**字号权保护要件**：

```
字号权保护 = 字号具有一定知名度 + 容易导致混淆
```

| 要件 | 说明 |
|------|------|
| 知名度 | 字号在相关公众中具有一定知名度 |
| 混淆可能性 | 容易导致相关公众混淆商品来源 |
| 行业关联 | 与商标核定使用的商品/服务有关联 |

**混淆的判断因素**：

| 因素 | 内容 |
|------|------|
| 商标/字号近似程度 | 文字构成、呼叫、含义 |
| 商品/服务类似程度 | 功能、用途、消费群体 |
| 字号知名度 | 在先使用时间、范围、宣传 |
| 主观意图 | 是否具有搭便车故意 |

> [!important] 突出使用
> 将与他人注册商标相同或者相近似的文字作为企业的字号在相同或者类似商品上**突出使用**，容易使相关公众产生误认的，构成商标侵权。

---

### 第6条　地理标志与商标冲突

> 原告以他人注册商标侵害其地理标志为由提起民事诉讼，符合民事诉讼法第一百一十九条规定的，人民法院应当受理。

**地理标志的保护**：

| 保护类型 | 说明 |
|----------|------|
| 未注册地理标志 | 作为在先权利保护 |
| 已注册集体/证明商标 | 依据商标法保护 |

**侵权认定**：

```
地理标志侵权 = 商品并非来源于该地区 + 误导公众
```

> [!warning] 要件
> - 诉争商标指定使用的商品与地理标志产品并非相同商品
> - 使用在该产品上仍然容易导致相关公众误认为该产品来源于该地区
> - 具有特定的质量、信誉或者其他特征

---

### 第7条　注册商标与在其他先权利冲突

> 原告以他人注册商标侵害其在先权益为由提起民事诉讼，符合民事诉讼法第一百一十九条规定的，人民法院应当受理。

**其他在先权益**：

| 权益类型 | 说明 |
|----------|------|
| 商品化权益 | 角色形象、名称的商品化权益 |
| 域名权益 | 域名与商标冲突 |
| 域名权 | 已经使用并具有一定影响的域名 |
| 其他民事权益 | 依民法典保护的合法权益 |

---

### 第8条　侵权认定规则

> 人民法院应当根据原告的诉讼请求和案件的具体情况，确定被告应当承担的民事责任。

**混淆/误认的判断标准**：

| 标准 | 内容 |
|------|------|
| 相关公众 | 与商品/服务相关的消费者和经营者 |
| 一般注意力 | 普通消费者通常的注意程度 |
| 隔离比对 | 分别观察，不同时比较 |
| 整体与部分 | 既要比对整体，也要比对主要部分 |

**举证要点**：

| 证据类型 | 证明内容 |
|----------|----------|
| 混淆证据 | 消费者混淆的实际证据 |
| 知名度证据 | 字号/商标的使用时间、范围 |
| 近似对比 | 标识之间的对比说明 |
| 主观恶意 | 被告攀附商誉的证据 |

---

### 第9条　民事责任承担

> 被告应当承担停止侵害、消除影响、赔偿损失等民事责任。

**责任承担方式**：

| 责任方式 | 适用情形 |
|----------|----------|
| 停止侵权 | 侵权行为持续进行 |
| 消除影响 | 侵权行为造成不良影响 |
| 赔偿损失 | 给原告造成经济损失 |
| 销毁商品 | 侵权商品及标识 |

**赔偿计算**：

| 计算方式 | 说明 |
|----------|------|
| 实际损失 | 原告因侵权所受损失 |
| 侵权获利 | 被告因侵权所获利益 |
| 许可费 | 参照许可使用费的合理倍数 |
| 法定赔偿 | 500万元以下 |

---

### 第10条　程序问题

> 涉及注册商标、企业名称与在先权利冲突的民事纠纷案件，由侵权行为实施地或者被告住所地人民法院管辖。

**管辖规则**：

| 要点 | 内容 |
|------|------|
| 管辖法院 | 侵权行为实施地、被告住所地法院 |
| 级别管辖 | 一般由中级人民法院管辖 |
| 证据保全 | 可以依申请进行证据保全 |
| 诉讼时效 | 3年（自知道或应当知道权利受侵害之日起） |

**当事人确定**：

| 当事人 | 确定规则 |
|--------|----------|
| 原告 | 在先权利人（企业、公民） |
| 被告 | 商标注册人/企业名称登记人 |
| 第三人 | 与案件有利害关系的人 |

## 📊 实务要点汇总

### 原告起诉要点

| 步骤 | 内容 |
|------|------|
| 1. 确定案由 | 侵害商标权纠纷 / 不正当竞争纠纷 |
| 2. 明确被告 | 商标注册人/企业名称登记人 |
| 3. 收集证据 | 在先权利证明、侵权证据、损害证据 |
| 4. 确定管辖 | 侵权行为地或被告住所地法院 |

### 被告抗辩要点

| 抗辩类型 | 要件 |
|----------|------|
| 在先权利不成立 | 原告权利不成立或不有效 |
| 不近似 | 标识不近似 |
| 不混淆 | 不会导致混淆 |
| 正当使用 | 描述性使用等正当理由 |
| 时效抗辩 | 超过诉讼时效 |

### 证据清单

| 类别 | 证据 |
|------|------|
| 权利证据 | 企业营业执照、名称核准文件、著作权登记证书、知名度证据 |
| 侵权证据 | 商标注册证、企业名称登记信息、实际使用证据 |
| 损害证据 | 经济损失证明、侵权获利证据、合理开支凭证 |
