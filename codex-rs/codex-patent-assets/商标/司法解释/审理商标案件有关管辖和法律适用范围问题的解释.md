# 最高人民法院关于审理商标案件有关管辖和法律适用范围问题的解释

> （2001年7月17日最高人民法院审判委员会第1189次会议通过，根据2020年12月23日最高人民法院审判委员会第1823次会议通过的《最高人民法院关于修改〈最高人民法院关于审理侵犯专利权纠纷案件应用法律若干问题的解释（二）〉等十八件知识产权类司法解释的决定》修正）

## 立法背景与目的

### 制定背景

2001年商标法修改后，商标纠纷案件类型更加复杂，需要明确：
- 各类商标案件的管辖法院
- 新旧法律适用的衔接
- 跨区域管辖的安排

### 规范目的

| 目的 | 说明 |
|------|------|
| 明确管辖 | 确定各级法院受理商标案件的范围 |
| 统一法律适用 | 解决新旧法衔接问题 |
| 方便诉讼 | 为当事人提供明确的诉讼指引 |

## 条款详解

### 第1条　商标民事纠纷案件范围

> 商标民事纠纷案件包括：

**案件类型**：

| 案件类型 | 说明 |
|----------|------|
| 侵害商标权纠纷 | 侵权行为认定、损害赔偿 |
| 商标合同纠纷 | 商标许可、转让合同纠纷 |
| 商标权属纠纷 | 商标权利归属争议 |
| 不正当竞争纠纷 | 与商标相关的不正当竞争 |
| 诉前禁令纠纷 | 行为保全申请 |

> [!note] 民事诉讼依据
> 商标民事纠纷案件依据《民事诉讼法》及其司法解释确定管辖。

---

### 第2条　商标行政案件范围

> 商标行政案件包括：

**案件类型**：

| 案件类型 | 说明 |
|----------|------|
| 不服驳回申请复审 | 商标局驳回复审决定 |
| 不服不予注册复审 | 异议复审决定 |
| 不服撤销/维持决定 | 撤销复审决定 |
| 不服无效宣告 | 无效宣告/复审决定 |
| 不服商标局决定 | 其他行政决定 |
| 商标授权确权行政诉讼 | 对上述决定的司法审查 |

> [!note] 行政诉讼依据
> 商标行政案件依据《行政诉讼法》及其司法解释确定管辖。

---

### 第3条　级别管辖

> 商标民事纠纷案件和商标行政案件由中级以上人民法院管辖。

**级别管辖**：

| 法院层级 | 管辖范围 |
|----------|----------|
| 基层人民法院 | 一般不受理商标案件 |
| 中级人民法院 | 管辖一审商标案件 |
| 高级人民法院 | 管辖本辖区重大案件 |
| 最高人民法院 | 管辖全国重大案件 |

> [!tip] 知识产权法院
> 北京、上海、广州知识产权法院对所在市的商标案件实行跨区域管辖。

---

### 第4条　地域管辖

> 侵害商标权纠纷案件，由侵权行为实施地、侵权商品的储藏地或者查封扣押地、被告住所地人民法院管辖。

**地域管辖**：

| 连接点 | 说明 |
|--------|------|
| 侵权行为实施地 | 侵权行为发生地 |
| 侵权商品储藏地 | 大量/经常性储存侵权商品地 |
| 查封扣押地 | 海关等行政机关查封扣押地 |
| 被告住所地 | 被告注册登记地 |

**管辖选择**：
- 原告可以向上述任一法院起诉
- 有多个管辖连接点时，原告有选择权

> [!important] 储藏地与查封扣押地
> - 储藏地：指大量或者经常性储存、隐匿侵权商品所在地
> - 查封扣押地：指海关等行政机关依法查封、扣押侵权商品所在地

---

### 第5条　商标授权确权案件管辖

> 商标授权确权行政案件由被告住所地的省、自治区、直辖市人民政府所在地的中级人民法院或者其他依法设立的中级人民法院管辖。

**行政诉讼管辖**：

| 案件类型 | 管辖法院 |
|----------|----------|
| 商标驳回复审 | 北京知识产权法院 |
| 商标异议复审 | 北京知识产权法院 |
| 商标无效宣告 | 北京知识产权法院 |
| 商标撤销复审 | 北京知识产权法院 |

> [!note] 专属管辖
> 商标授权确权行政案件由北京知识产权法院专属管辖。

---

### 第6条　专门法院管辖

> 知识产权法院和最高人民法院确定的基层人民法院管辖商标案件的范围，由最高人民法院确定。

**专门法院设置**：

| 法院 | 管辖区域 | 管辖范围 |
|------|----------|----------|
| 北京知识产权法院 | 北京市 | 商标授权确权行政案件 |
| 上海知识产权法院 | 上海市 | 商标民事纠纷案件 |
| 广州知识产权法院 | 广州市 | 商标民事纠纷案件 |

> [!tip] 跨区域管辖
> 知识产权法院对商标案件实行跨区域管辖。

---

### 第7条　法律适用范围

> 人民法院在审理商标案件时，对涉及商标法修改决定施行前发生的民事纠纷和行政诉讼案件，适用修改前的商标法；对涉及该决定施行后发生的民事纠纷和行政诉讼案件，适用修改后的商标法。

**法律适用规则**：

| 案件发生时间 | 适用法律 |
|--------------|----------|
| 修改决定施行前（2014年5月1日前） | 修改前商标法 |
| 修改决定施行后（2014年5月1日后） | 修改后商标法 |

> [!important] 2019年修改适用
> 同理，2019年商标法修改决定施行后的案件，适用2019年修正的商标法。

---

### 第8条　溯及力规定

> 涉及商标法修改决定施行前民事行为和行政行为发生的商标纠纷案件，适用修改前的商标法。

**溯及力规则**：

| 情形 | 适用法律 |
|------|----------|
| 侵权行为发生在施行日前 | 适用修改前商标法 |
| 侵权行为持续到施行日后 | 适用修改后商标法 |
| 授权确权决定在施行日前作出 | 适用修改前商标法 |

---

### 第9条　生效日期

> 本解释自公布之日起施行。

**施行日期**：
- 2002年1月21日公布施行
- 2020年12月29日修正（重新公布）

## 📊 实务要点汇总

### 诉讼管辖选择

| 案件类型 | 管辖法院选择 |
|----------|--------------|
| 商标侵权民事诉讼 | 侵权行为地/被告住所地中级法院 |
| 商标许可合同纠纷 | 被告住所地/合同履行地中级法院 |
| 商标行政诉讼 | 被告住所地（通常为北京）中级法院 |

### 管辖异议处理

| 情形 | 处理方式 |
|------|----------|
| 管辖错误 | 上诉至上一级法院 |
| 管辖权异议 | 应当在提交答辩状期间提出 |
| 移送管辖 | 审查后决定是否移送 |

### 管辖法院查询

| 步骤 | 内容 |
|------|------|
| 1 | 确定案件类型（民事/行政） |
| 2 | 确定级别（中级/高级/最高院） |
| 3 | 确定地域（侵权行为地/被告住所地） |
| 4 | 查证具体管辖法院 |
