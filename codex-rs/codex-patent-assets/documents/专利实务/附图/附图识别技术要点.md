# 技术图纸识别技术要点

## 图纸类型识别

### 化学结构式特征

- 苯环（六元环）、杂环结构
- 化学键（单键、双键、三键）
- 原子标记（C、N、O、S、P 等）
- SMILES 字符串表示
- 立体化学标记（楔形键、虚线键）

### 数学公式特征

- 积分符号 ∫
- 求和符号 Σ
- 希腊字母（α、β、γ 等）
- 矩阵表示
- 上下标

### 电学符号特征

- 电阻（R）、电容（C）、电感（L）
- 二极管、晶体管
- 逻辑门符号
- 接地符号
- 电源符号

### 通用机械图纸特征

- 尺寸标注线
- 剖面线
- 表面粗糙度符号
- 公差标注
- 装配关系

## OCR 识别要点

1. 优先识别图纸类型再进行结构化识别
2. 化学结构式需转为 SMILES 或 InChI
3. 数学公式需转为 LaTeX
4. 电学符号需转为标准电路符号集
5. 机械图纸需保留尺寸标注关系
