# 说明书附图规范

## 审查指南摘录

### 附图绘制要求

1. 附图应当使用制图工具和黑色墨水绘制，线条应当均匀清晰、足够深
2. 附图不得用铅笔、蜡笔、毛笔绘制，不得着色
3. 几幅图可以绘制在一张图纸上，但应当排列整齐、间隔清晰
4. 附图周围不得有框线

### 附图标记

- 附图标记应当使用阿拉伯数字
- 同一组成部分的附图标记在所有附图中应当一致
- 附图标记不得用于说明书中未提及的组成部分

### 流程图规范

- 使用标准流程图符号（矩形=处理、菱形=判断、平行四边形=输入输出）
- 流程方向一般从上到下、从左到右
- 各步骤间用带箭头的连线连接

### 电路图规范

- 使用标准电气符号
- 元件编号应与说明书一致
- 连接关系应清楚明确

### 化学结构式

- 使用标准化学键表示
- 原子符号应符合化学命名规范
- 立体化学应清楚标注

## 附图说明撰写

### 格式要求

说明书中的附图说明部分应按照以下格式撰写：

```
图1是本发明实施例提供的XXX系统的结构示意图；
图2是本发明实施例提供的XXX方法的流程图；
图3是本发明实施例提供的XXX装置的结构框图。
```

### 注意事项

1. 每幅图对应一段说明
2. 说明应简要概括附图内容
3. 不应在附图说明中包含技术特征的具体描述
4. 图号应连续编号
