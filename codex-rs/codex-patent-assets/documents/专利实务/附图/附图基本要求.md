# 专利附图基本要求

## 法律依据

- 专利法实施细则第18条：说明书附图
- 审查指南第一部分第二章 7. 说明书的附图
- 审查指南第二部分第二章 2.3 说明书附图

## 附图类型

### 发明和实用新型

1. **流程图**：方法权利要求的步骤流程
2. **框图**：系统或装置的功能模块
3. **电路图**：电子电路连接关系
4. **结构示意图**：产品各部件的空间关系
5. **剖视图**：内部结构的截面表示
6. **状态图**：状态转换关系

### 外观设计

1. **六面正投影视图**：主视图、后视图、左视图、右视图、俯视图、仰视图
2. **立体图（使用状态参考图）**：可选

## 附图标记规范

1. 附图标记使用阿拉伯数字编号
2. 同一部件在不同视图中使用相同标记
3. 标记应标注在部件附近，不与图线交叉
4. 说明书中提及的附图标记必须在附图中出现

## 常见问题

1. **附图不清楚**：线条模糊、分辨率不足
2. **缺少必要附图**：说明书提及"如图X所示"但无对应附图
3. **标记不一致**：说明书与附图标记不对应
4. **尺寸比例不当**：无法清楚表达发明内容
