# 商标复审 Wiki 卡片

> **本卡片专注于商标复审程序，包括驳回复审、异议复审、撤销复审等各类复审流程**

---

## 📋 基本信息

| 项目 | 内容 |
|------|------|
| **定义** | 商标复审是指当事人不服商标局做出的决定，依法向商标评审委员会申请重新审查的程序 |
| **法律依据** | 《中华人民共和国商标法》（2019年修正）、《中华人民共和国商标法实施条例》（2014年） |
| **复审机构** | 国家知识产权局商标评审委员会 |
| **后续救济** | 对复审决定不服可向北京知识产权法院提起行政诉讼 |

---

## 🔄 商标复审类型概览

### 四种复审类型对比

| 复审类型 | 适用情形 | 申请期限 | 审理时限 | 申请资格 |
|---------|---------|---------|---------|---------|
| **驳回复审** | 对商标局驳回申请不服 | 收到通知之日起**15日内** | **9个月**（可延长3个月） |商标注册申请人 |
| **不予注册复审** | 对商标局不予注册决定不服 | 收到通知之日起**15日内** | **12个月**（可延长6个月） | 被异议人 |
| **无效宣告复审** | 对商标局宣告无效决定不服 | 收到通知之日起**15日内** | **9个月**（可延长3个月） | 商标注册人 |
| **撤销复审** | 对商标局撤销决定不服 | 收到通知之日起**15日内** | **9个月**（可延长3个月） | 商标注册人 |

## 🎯 重点：商标异议复审流程详解

### 一、商标异议程序（前置程序）

#### 1. 异议提起

**法律依据：商标法第三十三条**

> 对初步审定公告的商标，自公告之日起**三个月内**：
>
> - **在先权利人、利害关系人**认为违反本法第十三条第二款和第三款、第十五条、第十六条第一款、第三十条、第三十一条、第三十二条规定的，可以向商标局提出异议
> - **任何人**认为违反本法第四条、第十条、第十一条、第十二条、第十九条第四款规定的，可以向商标局提出异议

**要点解读**：

| 异议主体 | 可依据的条款 | 保护对象 |
|---------|------------|---------|
| **在先权利人、利害关系人** | 第十三条、第十五条、第十六条、第三十条、第三十一条、第三十二条 | 特定民事权益 |
| **任何人** | 第四条、第十条、第十一条、第十二条、第十九条第四款 | 公共利益 |

**异议期限**：
- 初步审定公告之日起**3个月内**
- 不得延期

**异议费用**：750元/件

#### 2. 商标局异议审查

**法律依据：商标法第三十五条第一款**

> 商标局应当听取异议人和被异议人陈述事实和理由，经调查核实后，自公告期满之日起**十二个月**内做出是否准予注册的决定，并书面通知异议人和被异议人。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长**六个月**。

**审查流程**：

```
1. 收到异议申请
   ├─ 形式审查
   ├─ 受理并通知被异议人
   └─ 被异议人答辩（15日内）

2. 实质审查
   ├─ 听取双方陈述
   ├─ 证据交换
   └─ 调查核实

3. 做出决定
   ├─ 准予注册
   └─ 不予注册
```

**审查结果**：

| 结果 | 后果 | 救济途径 |
|------|------|---------|
| **准予注册** | 发给商标注册证，并予公告 | 异议人可请求宣告无效（第44条） |
| **不予注册** | 书面通知异议人和被异议人 | 被异议人可申请复审（第35条） |

### 二、不予注册复审（异议复审）

#### 1. 复审申请

**法律依据：商标法第三十五条第二款**

> 商标局做出不予注册决定，**被异议人不服的**，可以自收到通知之日起**十五日内**向商标评审委员会申请复审。

**申请资格**：
- **仅限被异议人**（商标注册申请人）
- 异议人无权申请不予注册复审
- 如异议人对准予注册决定不服，可请求宣告无效（第44条）

**申请期限**：
- 收到商标局不予注册决定之日起**15日内**
- **不得延期**
- 逾期视为接受决定

**申请费用**：750元/件

#### 2. 商标评审委员会复审

**法律依据：商标法第三十五条第二款**

> 商标评审委员会应当自收到申请之日起**十二个月**内做出复审决定，并书面通知异议人和被异议人。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长**六个月**。

**审理时限**：
- 法定时限：**12个月**
- 可延长：**6个月**
- 最长：**18个月**

**审理流程**：

```
1. 申请阶段
   ├─ 提交复审申请书
   ├─ 缴纳复审费用
   └─ 提交证据材料

2. 受理阶段
   ├─ 形式审查
   ├─ 受理或补正通知
   └─ 送达对方当事人

3. 审理阶段
   ├─ 对方答辩（收到通知后30日内）
   ├─ 证据交换
   ├─ （可能）口头审理
   └─ 案件审理

4. 决定阶段
   ├─ 制作复审决定书
   ├─ 送达双方当事人
   └─ 决定文书公开
```

**审理结果**：

| 结果 | 后果 | 救济途径 |
|------|------|---------|
| **维持不予注册决定** | 商标不予注册，不发给注册证 | 双方均不服可起诉（30日内） |
| **撤销不予注册决定** | 商标予以注册，发给注册证 | 异议人若不服可请求宣告无效 |

#### 3. 司法救济

**法律依据：商标法第三十五条第二款**

> 被异议人对商标评审委员会的决定不服的，可以自收到通知之日起**三十日内**向人民法院起诉。人民法院应当通知**异议人作为第三人**参加诉讼。

**诉讼要点**：

| 项目 | 内容 |
|------|------|
| **管辖法院** | 北京知识产权法院（一审） |
| **起诉期限** | 收到复审决定之日起**30日内** |
| **诉讼性质** | 行政诉讼（被告：国家知识产权局） |
| **第三人** | 异议人作为第三人参加诉讼 |
| **二审** | 不服一审判决可上诉至北京市高级人民法院 |

**特别说明**：
- **被异议人**可以提起行政诉讼
- **异议人**作为第三人参加诉讼
- 双方当事人均可对法院判决提起上诉

### 三、异议复审的特殊情形

#### 1. 异议人请求宣告无效

**法律依据：商标法第三十五条第二款**

> 商标局做出准予注册决定的，发给商标注册证，并予公告。**异议人不服的**，可以依照本法第四十四条、第四十五条的规定向商标评审委员会请求宣告该注册商标无效。

**要点**：
- 异议人**不能申请不予注册复审**
- 只能依据**第四十四条**（绝对理由）或**第四十五条**（相对理由）请求宣告无效
- 时间限制：
  - 第四十四条：无时间限制
  - 第四十五条：自商标注册之日起5年内（恶意驰 名商标除外）

#### 2. 在先权利确定的中止

**法律依据：商标法第三十五条第三款**

> 商标评审委员会在依照前款规定进行复审的过程中，所涉及的在先权利的确定必须以人民法院正在审理或者行政机关正在处理的另一案件的结果为依据的，**可以中止审查**。中止原因消除后，应当恢复审查程序。

**中止情形**：
- 在先权利正在法院审理中
- 在先商标正在撤销或无效程序中
- 在先权利（如著作权、姓名权）的确定需要等待其他案件结果

## 📝 驳回复审（简要说明）

### 申请条件

**法律依据：商标法第三十四条**

> 对驳回申请、不予公告的商标，商标局应当书面通知商标注册申请人。商标注册申请人不服的，可以自收到通知之日起**十五日内**向商标评审委员会申请复审。

**审理时限**：**9个月**（可延长3个月）

**常见驳回理由及复审策略**：

| 驳回理由 | 复审策略 | 证据准备 |
|---------|---------|---------|
| **与在先商标近似** | 主张不近似、区分商品/服务 | 商标使用证据、知名度证据 |
| **缺乏显著性** | 主张经过使用获得显著性 | 长期使用证据、广告宣传证据 |
| **不良影响** | 主张不构成不良影响 | 商标设计理念、市场反馈 |

## 📌 申请材料要求

### 1. 复审申请书（必交）

根据《关于简化驳回复审案件申请材料的公告》（2018年），复审申请书应包括：

- **申请人信息**：名称、地址、联系方式
- **被申请人信息**：名称、地址
- **复审请求**：明确请求撤销原决定的具体内容
- **事实和理由**：详细陈述法律理由和事实依据
- **证据清单**：列明提交的证据材料

### 2. 证据材料（必交）

**不予注册复审**应提交：
- 商标局不予注册决定书
- 初步审定公告页
- 商标使用证据（如有）
- 知名度证据（如有）
- 其他支持复审请求的证据

**证据要求**：
- 证据应与复审请求密切相关
- 证据应制作清单并标明页码
- 外文证据应附中文译文

### 3. 其他材料

- **委托代理机构**：需提交代理委托书
- **身份证明**：企业营业执照复印件或身份证复印件
- **复审费用缴费凭证**

## ⏱️ 重要时限提醒

### 异议程序时限

| 事项 | 时限 | 依据 |
|------|------|------|
| **提出异议** | 公告期满前3个月内 | 第33条 |
| **被异议人答辩** | 收到异议通知之日起30日内 | 实施条例第27条 |
| **商标局异议审查** | 公告期满之日起12个月（可延长6个月） | 第35条 |

### 复审程序时限

| 事项 | 时限 | 依据 |
|------|------|------|
| **申请不予注册复审** | 收到不予注册决定之日起15日内 | 第35条 |
| **被申请人答辩** | 收到答辩通知之日起30日内 | 实施条例 |
| **评审委员会审理** | 收到申请之日起12个月（可延长6个月） | 第35条 |
| **提起行政诉讼** | 收到复审决定之日起30日内 | 第35条 |

**⚠️ 特别注意**：
- **15日申请期限不可延期**，逾期将失去复审机会
- **30日起诉期限不可延期**，逾期复审决定生效
- 期限最后一日是法定节假日的，以节假日后第一个工作日为准

## 🎯 实务策略与建议

### 1. 异议阶段的策略

**对于被异议人（商标申请人）**：

| 情形 | 策略 |
|------|------|
| **异议理由充分** | 考虑协商，避免冗长程序 |
| **异议理由薄弱** | 积极答辩，准备复审 |
| **商标价值重大** | 全力应诉，准备复审+诉讼 |

**答辩要点**：
- 及时提交答辩书（30日内）
- 针对异议理由逐条反驳
- 提交商标使用证据
- 提交知名度证据（如适用）
- 强调商标的显著性

### 2. 复审申请的策略

**申请前评估**：

```
是否申请不予注册复审？
│
├─ 评估不予注册决定的法律依据是否正确
│
├─ 评估商标的商业模式价值
│
├─ 评估复审胜诉的可能性
│
├─ 评估时间和经济成本
│
└─ 做出决策
```

**申请策略**：

| 情形 | 建议 |
|------|------|
| **决定适用法律明显错误** | 强烈建议申请复审 |
| **事实认定不清** | 建议申请复审，补充证据 |
| **商标使用价值巨大** | 建议申请复审 |
| **法律和事实均不利** | 考虑放弃，重新申请 |

### 3. 复审审理的策略

**书面审理阶段**：
- 提交详细的复审理由书
- 提交充分的证据材料
- 针对对方答辩进行反驳

**口头审理**（如举行）：
- 准备详细的代理意见
- 准备证据原件
- 提前演练陈述要点
- 注意法庭礼仪

### 4. 行政诉讼的策略

**起诉前评估**：
- 复审决定的法律适用是否正确
- 事实认定是否清楚
- 程序是否合法
- 胜诉可能性

**诉讼策略**：
- 明确诉讼请求
- 针对复审决定的错误提起诉讼
- 可以提交新证据
- 注意诉讼时效（30日）

## 🔍 常见问题FAQ

### Q1：异议人能否申请不予注册复审？

**答**：**不能**。不予注册复审的申请权仅属于被异议人（商标注册申请人）。如果异议人对商标局准予注册的决定不服，只能依据第四十四条或第四十五条请求宣告该注册商标无效。

### Q2：不予注册复审和无效宣告有什么区别？

**答**：

| 对比维度 | 不予注册复审 | 无效宣告 |
|---------|------------|---------|
| **申请主体** | 仅限被异议人 | 在先权利人、利害关系人、任何人 |
| **申请时机** | 商标不予注册决定后 | 商标已注册后 |
| **法律依据** | 第三十五条 | 第四十四条、第四十五条 |
| **申请期限** | 收到决定后15日内 | 45条：5年内；44条：无限制 |

### Q3：商标异议期间，商标能否转让？

**答**：异议期间商标申请权可以转让，但受让人将承继原申请人的地位。如果被异议，复审申请权一并转移。

### Q4：复审期间能否提交新证据？

**答**：
- **书面审理阶段**：原则上可以提交，但应有正当理由
- **口头审理前**：可提交，需给对方质证时间
- **行政诉讼阶段**：可以提交新证据

### Q5：复审决定何时生效？

**答**：
- **双方均不起诉**：复审决定自送达之日起**30日后**生效
- **一方提起诉讼**：复审决定不生效，等待法院判决
- **法院判决**：一审判决或二审判决生效时作为最终结果

### Q6：复审中能否请求认定引证商标无效？

**答**：**不能在同一申请中提出，但可以分别申请并申请中止审理。** 详细说明见 [[复审中请求引证商标无效FAQ]]

## 📌 核心要点总结

### 异议复审流程图

```
商标初步审定公告
        ↓
   [3个月异议期]
        ↓
     ┌───┴───┐
     ↓       ↓
   有异议   无异议
     ↓       ↓
 商标局   →  予以注册
 异议审查
     ↓
 ┌───┴───┐
 ↓       ↓
准予注册  不予注册
 ↓          ↓
 异议人   被异议人
 可请求   可申请
无效宣告  不予注册复审
           ↓
       商标评审委员会
       [12个月审理]
           ↓
       ┌───┴───┐
       ↓       ↓
     维持   撤销原决定
     原决定       ↓
       ↓      予以注册
    被异议人
    可起诉
    [30日内]
```

### 关键时间节点

| 时间节点 | 时限 | 逾期后果 |
|---------|------|---------|
| 提出异议 | 公告期满前3个月内 | 丧失异议权 |
| 被异议人答辩 | 收到通知后30日内 | 视为放弃答辩 |
| 申请复审 | 收到决定后15日内 | **丧失复审权** |
| 提起诉讼 | 收到决定后30日内 | 复审决定生效 |

### 成功要诀

✅ **严格把握时限**：15日复审期限绝对不可延期

✅ **证据准备充分**：商标使用证据、知名度证据、在先权利证据

✅ **理由陈述清晰**：针对原决定的错误逐条反驳

✅ **专业代理协助**：建议委托专业知识产权律师或代理机构
