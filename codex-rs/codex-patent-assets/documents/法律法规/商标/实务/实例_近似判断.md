## 近似判断核心原则

### 判断主体标准

> 以**相关公众的一般注意力**为标准

**相关公众**指：
- 与商标所标识的商品/服务有关的消费者
- 与前述商品/服务的营销有密切关系的其他经营者

### 隔离比对原则

> 既要进行商标的**整体比对**，又要进行商标的**主要部分**比对，比对应当在比对对象**隔离的状态**下分别进行

### 显著性和知名度

> 判断商标是否近似，应当考虑请求保护注册商标的**显著性和知名度**

## 文字商标近似判断

### 近似要素分析

| 要素 | 判断内容 |
|------|----------|
| 字形 | 文字的书写形式、笔画、结构 |
| 读音 | 文字的发音是否相同或近似 |
| 含义 | 文字表达的意义是否相同或近似 |

### 近似判断示例

```
示例1："华为" vs "华韦"
- 字形：近似（华字相同）
- 读音：近似（声母相同）
- 含义：可区分
→ 判断：近似

示例2："苹果" vs "苹果酱"
- 字形：相同
- 读音：相同
- 含义：不同（酱为商品类型）
→ 判断：不近似（含义可区分）
```

## 组合商标近似判断

### 判断方法

1. **分别比对**：文字部分、图形部分分别比对
2. **组合比对**：作为整体进行比对
3. **隔离观察**：在隔离状态下分别观察

### 近似判断规则

```
组合商标近似 = 文字近似 + 图形近似（任一要素近似）
                或
                整体印象近似
```

|------|----------|

## 类似商品判断（页码290）

### 判断因素

| 因素 | 说明 |
|------|------|
| 功能用途 | 商品的实际用途是否相同 |
| 生产部门 | 生产制造是否相同/关联 |
| 销售渠道 | 销售途径是否相同 |
| 消费对象 | 目标消费群体是否相同 |

### 近似判断与商品类似的交互

> 商标近似 + 商品类似 = 侵权可能性高
> 商标近似 + 商品不类似 = 侵权可能性低（除非是驰 名商标）
> 商标不近似 + 商品类似 = 一般不构成侵权

## 📚 审查依据

- **《商标法》第30条**：申请注册的商标不得与他人在先商标近似
- **《商标法》第13条**：驰 名商标跨类保护
- **《审理商标民事纠纷案件适用法律若干问题的解释》第9-12条**：近似判断标准
