# 商标复审中能否请求认定引证商标无效？

> 本卡片解答商标复审程序中关于引证商标无效宣告的常见问题，包括程序选择、策略建议和操作要点。

---

## 🎯 核心答案

**不能在同一复审申请中同时提出，但可以分别申请并申请中止审理。**

---

## 📋 详细解答

### 一、复审与无效宣告是两个独立程序

#### 1. 程序性质不同

| 对比维度 | 复审程序 | 无效宣告程序 |
|---------|---------|-------------|
| **法律性质** | 对商标局决定不服的**救济程序** | 对已注册商标提出的**确权程序** |
| **审查对象** | 申请商标是否应予注册 | 引证商标是否应无效 |
| **申请主体** | 被驳回/被异议的商标申请人 | 在先权利人、利害关系人、任何人 |
| **法律依据** | 商标法第34条、第35条等 | 商标法第44条、第45条 |
| **申请书式** | 复审申请书 | 无效宣告申请书 |

#### 2. 审理原则不同

**复审程序**：
- 审查申请商标是否符合注册条件
- 引证商标的有效性是**既定前提**
- 一般不审理引证商标是否应无效

**无效宣告程序**：
- 专门审理已注册商标是否符合无效条件
- 需要另行组成合议组
- 审理引证商标是否符合注册条件

---

### 二、实务操作方案

#### 方案一：分别申请 + 申请中止（推荐）

**操作步骤**：

```
步骤1：提起复审
   ├─ 在15日内提交复审申请
   ├─ 缴纳复审费用（750元）
   └─ 提交复审材料

步骤2：对引证商标提起无效宣告
   ├─ 另行提交无效宣告申请
   ├─ 缴纳无效宣告费用（750元）
   └─ 提交引证商标无效的证据

步骤3：申请复审中止审理
   ├─ 在规定期限内提出中止申请
   ├─ 说明已对引证商标提起无效宣告
   └─ 等待商评委决定是否中止

步骤4：等待无效结果
   ├─ 引证商标被宣告无效 → 复审程序恢复
   └─ 引证商标维持有效 → 复审程序继续
```

**中止申请要点**：

根据《商标评审中止情形规范》解读：

| 要点 | 内容 |
|------|------|
| **中止原则** | 以**必要为原则** - 只有在先权利确定对审理结果有实质性影响时才中止 |
| **申请期限** | 驳回复审案件：提交复审申请后**3个月补正期限内** |
| **审查标准** | 商评委根据必要性原则自行裁量决定是否中止 |
| **中止期限** | 并非无限期中止 - 一般等待无效宣告程序终结 |

#### 方案二：只申请复审（不建议）

**风险**：
- 如果引证商标确实存在无效情形，但未提起无效宣告
- 商评委可能会维持驳回决定
- 错失消除注册障碍的机会

---

### 三、法律依据

#### 1. 商标法关于中止的规定

**商标法第三十五条第四款**（适用于异议复审，同样适用于其他复审）：

> 商标评审委员会在依照前款规定进行复审的过程中，所涉及的**在先权利的确定**必须以人民法院正在审理或者行政机关正在处理的另一案件的结果为依据的，**可以中止审查**。中止原因消除后，应当恢复审查程序。

**要点**：
- 在先权利（如引证商标权）的确定需要等待其他案件结果
- 商评委**可以**（而非必须）中止审查
- 是否中止由商评委根据必要性原则裁量

#### 2. 商标评审规则关于程序独立的规定

**商标评审规则（2014年修订）**相关规定：

- **第三十条**：经不予注册复审程序予以核准注册的商标，原异议人向商标评审委员会请求无效宣告的，商标评审委员会应当**另行组成合议组**进行审理。

**解读**：
- 复审程序和无效宣告程序是**独立**的
- 需要不同的合议组审理
- 不能在同一程序中混同

#### 3. "一事不再理"原则

根据商标评审实践：

> 商标评审委员会对商标评审申请已经作出裁定或者决定的，任何人不得以**相同的事实和理由**再次提出评审申请。

**解读**：
- 复审和无效宣告的**事实和理由不同**
- 复审：申请商标应注册
- 无效宣告：引证商标应无效
- 因此不违反"一事不再理"原则

### 四、具体操作指南

#### 1. 何时分别申请

**适合分别申请的情形**：

| 情形 | 说明 |
|------|------|
| **引证商标明显可无效** | 如引证商标连续3年未使用、恶意注册等 |
| **引证商标是主要障碍** | 消除引证商标障碍后，申请商标可能获注册 |
| **有时间准备** | 有足够时间准备两个申请的材料 |
| **有预算支持**| 能够承担两个申请的费用（共1500元） |

#### 2. 申请时间安排

**推荐时间节点**：

```
收到驳回/不予注册决定（Day 0）
        ↓
    Day 1-10：准备复审材料
        ↓
    Day 11：提交复审申请
        ↓
    Day 11-13：准备无效宣告材料
        ↓
    Day 14-15：提交无效宣告申请
        ↓
    Day 16-45：提交中止申请
```

**注意事项**：
- ⚠️ 复审申请必须在**15日内**提交，不可延期
- ⚠️ 无效宣告申请虽无时间限制，但**建议尽早提交**
- ⚠️ 中止申请应在**补正期限内**（通常3个月内）提出

#### 3. 中止申请书要点

**中止申请书应包括**：

1. **基本信息**：
   - 复审案件编号
   - 申请人信息
   - 引证商标信息

2. **中止理由**：
   - 已对引证商标提起无效宣告申请
   - 无效宣告案件编号
   - 引证商标有效性是复审案件的先决条件

3. **必要性说明**：
   - 引证商标无效若成立，复审障碍消除
   - 等待无效结果有利于案件审理
   - 符合必要性和经济性原则

4. **附件**：
   - 无效宣告申请受理通知书
   - 引证商标无效证据摘要

### 五、风险提示与建议

#### 1. 不能保证中止

**重要提示**：
- ⚠️ 商评委**有权决定是否中止**，并非申请即中止
- ⚠️ 即使中止，也可能在无效结果出来前恢复审理
- ⚠️ 中止审查并非无限期延长

#### 2. 时间成本

| 项目 | 时间 |
|------|------|
| 复审审理 | 9-12个月 |
| 无效宣告审理 | 12-18个月 |
| 中止等待 | 视情况而定 |

**总计**：可能长达**2-3年**

#### 3. 费用成本

| 项目 | 费用（元） |
|------|-----------|
| 复审申请 | 750 |
| 无效宣告申请 | 750 |
| 可能的行政诉讼 | 额外费用 |
| **合计** | **至少1500元** |

#### 4. 策略建议

| 情形 | 建议 |
|------|------|
| **强案例**（引证商标明显可无效） | ✅ 分别申请 + 申请中止 + 等待无效结果 |
| **弱案例**（无效理由不足） | ⚠️ 专注复审答辩，考虑其他策略（撤三、协商等） |
| **时间紧迫** | ⚠️ 优先确保复审申请在15日内提交，无效宣告可稍后补充 |
| **预算有限** | ⚠️ 选择最有把握的程序，优先考虑胜算高的路径 |

### 六、典型案例分析

#### 案例1：成功中止案例

**案情**：
- 申请商标被驳回
- 引证商标连续3年未使用
- 申请人在复审期间对引证商标提起撤三
- 同时提起无效宣告（恶意注册）

**结果**：
- 商评委中止复审审理
- 引证商标被撤销
- 复审程序恢复，申请商标获注册

#### 案例2：未中止案例

**案情**：
- 申请商标被驳回
- 申请人对引证商标提起无效宣告
- 申请中止审理

**结果**：
- 商评委认为中止必要性不足
- 复审程序继续进行
- 维持驳回决定

**原因**：
- 引证商标无效理由不充分
- 即使不中止，复审案件也可审理
- 未达到"实质性影响"标准

### 七、常见问题FAQ

#### Q1：复审和无效宣告能否用同一份申请书？

**答**：**不能**。复审和无效宣告是两个独立程序，需要：
- 分别提交申请书
- 分别缴纳费用（各750元）
- 使用不同的申请书式

#### Q2：无效宣告申请有期限限制吗？

**答**：
- **相对理由无效**（第45条）：自商标注册之日起**5年内**
- **绝对理由无效**（第44条）：**无时间限制**
- 恶意注册的驰 名商标：**无时间限制**

#### Q3：如果引证商标被无效，复审决定会改变吗？

**答**：会。
- 引证商标被无效后，复审障碍消除
- 商评委重新审理
- 可能撤销原驳回决定，准予注册

#### Q4：能否在复审答辩书中提及引证商标应无效？

**答**：
- ✅ 可以在**理由部分**提及
- ✅ 说明引证商标存在无效情形
- ✅ 但不能作为复审请求提出
- ⚠️ 仍需另行提交无效宣告申请

#### Q5：中止申请被驳回怎么办？

**答**：
- 复审程序继续进行
- 但无效宣告程序仍在进行
- 如果无效成功，可考虑重新申请或启动其他程序

#### Q6：能否对多个引证商标同时提起无效宣告？

**答**：**可以**。
- 对每个引证商标分别提起无效宣告
- 在中止申请中一并说明
- 商评委会综合考虑

---

## ✅/❌ 要点总结

### ✅ 可以做：

1. 在复审期间**单独**对引证商标提起无效宣告
2. 申请复审程序**中止审理**
3. 等待无效宣告结果后再继续复审

### ❌ 不能做：

1. 在复审申请书中**同时请求**认定引证商标无效
2. 用复审申请替代无效宣告申请
3. 在同一程序中混同两个不同请求

### 💡 策略建议：

| 情形 | 建议 |
|------|------|
| 强案例（引证商标明显可无效） | 分别申请 + 申请中止 |
| 弱案例（无效理由不足） | 专注复审答辩 |
| 时间紧迫 | 先提交复审，后补充无效宣告 |
| 预算有限 | 选择最有把握的程序 |
