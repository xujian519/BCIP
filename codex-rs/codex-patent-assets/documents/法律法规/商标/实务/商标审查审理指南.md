## 🎯 核心审查标准

### 1. 商标显著性判断

**不具有显著性的情形**（商标法第11条）：

| 情形 | 说明 | 例外 |
|------|------|------|
| 仅有商品通用名称 | 行业通用名称 | 可通过使用获得显著性 |
| 仅描述商品特点 | 质量、原料、功能等 | 独特表达方式可注册 |
| 其他缺乏特征 | 过于简单或复杂 | 独创设计可注册 |

**获得显著性的认定**：
- 长期使用（通常3年以上）
- 大量广告宣传
- 市场份额
- 相关公众认知

### 2. 商标近似判断

**判断原则**：

| 原则 | 说明 |
|------|------|
| 相关公众标准 | 以普通消费者的注意力为判断标准 |
| 隔离比对原则 | 分离观察，不同时比较 |
| 整体与部分 | 既要比对整体，也要比对主要部分 |
| 显著性与知名度 | 考虑商标的显著性和知名度 |

**近似判断要素**：

| 要素 | 内容 |
|------|------|
| 文字近似 | 字形、读音、含义近似 |
| 图形近似 | 构图、颜色、整体外观近似 |
| 组合近似 | 各要素组合后整体近似 |

### 3. 商品类似判断

**类似商品**：

```
类似商品 = 功能 + 用途 + 生产部门 + 销售渠道 + 消费对象
```

| 判断因素 | 说明 |
|----------|------|
| 功能 | 商品的实际用途 |
| 用途 | 商品的使用目的 |
| 生产部门 | 制造商是否相同 |
| 销售渠道 | 销售途径是否相同 |
| 消费对象 | 目标消费群体是否相同 |

## 🔍 典型实例类型

### 类型一：文字商标近似

**实例位置**：第271-280页

**判断标准**：
- 字形是否近似
- 呼叫是否近似
- 含义是否近似
- 整体是否近似

**典型案例**：
- "华为" vs "华韦" → 近似
- "小米" vs "小觅" → 不近似
- "苹果" vs "苹果酱" → 不近似

### 类型二：图形商标近似

**实例位置**：第281-295页

**判断标准**：
- 构图是否近似
- 颜色组合是否近似
- 整体外观是否近似
- 是否有区别性特征

**典型案例**：
- 几何图形组合相似 → 近似
- 写实图形 vs 抽象图形 → 不近似

### 类型三：商品类似判断

**实例位置**：第296-310页

**判断标准**：
- 功能用途是否相同
- 生产销售是否关联
- 消费群体是否重叠

**典型案例**：
- 服装 vs 鞋帽 → 类似
- 手机 vs 电脑 → 类似
- 饮料 vs 食品 → 不类似
