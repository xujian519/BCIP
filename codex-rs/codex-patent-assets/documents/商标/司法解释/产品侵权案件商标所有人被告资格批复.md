# 最高人民法院关于产品侵权案件的受害人能否以产品的商标所有人为被告提起民事诉讼的批复

> （2002年7月11日最高人民法院审判委员会第1229次会议通过，根据2020年12月23日最高人民法院审判委员会第1823次会议通过的《最高人民法院关于修改〈最高人民法院关于审理侵犯专利权纠纷案件应用法律若干问题的解释（二）〉等十八件知识产权类司法解释的决定》修正）

## 问题提出

### 背景

在产品责任纠纷中，存在一个重要的法律问题：

```
法律问题：
产品的商标所有人（如品牌授权方）是否应当对产品缺陷导致的侵权承担民事赔偿责任？
```

### 实践中的争议

| 观点 | 理由 |
|------|------|
| 认为可以起诉 | 商标所有人从商标授权中获益，应承担责任 |
| 认为不能起诉 | 商标所有人不是产品生产者，不承担产品责任 |
| 认为需要具体分析 | 视具体情况确定责任主体 |

---

## 法律意见

### 最高人民法院答复

> 最高人民法院答复：不能。

**核心原则**：

```
产品责任（产品缺陷侵权）≠ 商标侵权

产品商标所有人 ≠ 产品生产者
```

**理由**：

| 理由 | 说明 |
|------|------|
| 主体分离 | 商标所有人（品牌方）与产品生产者（制造商）是不同主体 |
| 责任主体 | 产品责任由产品生产者承担，不是商标所有人 |
| 获益关系 | 商标授权使用与产品缺陷没有直接因果关系 |
| 法律依据 | 产品责任适用产品责任法，不是商标法 |

> [!important] 基本原则
> **产品的商标所有人不承担产品缺陷导致的侵权责任**，除非有例外情形。

---

## 适用条件

### 例外情形

虽然基本原则是不可以起诉商标所有人，但存在例外情形时可以突破：

**例外情形一：商标所有人与生产者为同一人**

```
情形：商标所有权人同时也是产品生产者
处理：可以作为被告

示例：A公司既是商标注册人，又是自己生产产品的制造商
```

**例外情形二：商标所有人参与产品生产或设计**

```
情形：商标所有人对产品缺陷存在过错
处理：可能承担过错责任

示例：
- 商标所有人参与了产品的设计
- 商标所有人指定了产品的原材料
- 商标所有人参与了产品的质量控制
```

**例外情形三：商标许可合同中的特殊约定**

```
情形：许可合同约定了产品质量责任承担
处理：依约确定责任

示例：许可合同中明确约定了商标所有人对产品质量承担责任
```

**例外情形四：挂牌生产（定牌加工）**

```
情形：商标所有人委托他人生产并挂牌
处理：需要具体分析

示例：
- 受托方有生产资质，商标所有人不承担产品责任
- 商标所有人对受托方生产进行实质控制，可能承担责任
```

## 实务意义

### 诉讼主体资格判断

| 情形 | 被告资格 | 说明 |
|------|----------|------|
| 商标所有人=生产者 | ✅ 可以 | 同一主体 |
| 商标所有人≠生产者 | ❌ 不可以 | 产品责任由生产者承担 |
| 商标所有人有过错 | ✅ 可以 | 依过错程度承担责任 |
| 存在许可合同约定 | ✅ 可以 | 依约确定 |

### 当事人诉讼策略

**对原告（受害人）**：

| 步骤 | 内容 |
|------|------|
| 1 | 确定产品生产者（制造商） |
| 2 | 确认产品是否存在缺陷 |
| 3 | 证明损害与产品缺陷的因果关系 |
| 4 | 如要起诉商标所有人，需证明例外情形存在 |

**对被告（商标所有人）**：

| 要点 | 说明 |
|------|------|
| 主体抗辩 | 主张与产品生产者不是同一主体 |
| 责任抗辩 | 产品责任应由生产者承担 |
| 过错抗辩 | 对产品缺陷不存在过错 |

### 证据准备

**原告需要准备的证据**：

| 证据类型 | 证明内容 |
|----------|----------|
| 产品来源证据 | 产品由谁生产 |
| 商标权属证据 | 商标所有权归属 |
| 缺陷证据 | 产品存在缺陷 |
| 损害证据 | 实际损害结果 |
| 因果关系证据 | 缺陷与损害的因果关系 |

**例外情形需要准备的证据**：

| 例外情形 | 需要的证据 |
|----------|----------|
| 同为生产者 | 生产资质、营业执照等 |
| 参与生产设计 | 生产参与证据、质量控制记录 |
| 合同约定 | 许可合同、补充协议 |

## 典型案例类型

### 案例类型一：品牌授权生产

**案情**：
A公司将其商标授权B公司使用，B公司生产的产品存在缺陷导致消费者C受伤。

**裁判结果**：
- C只可以起诉B公司（生产者）
- C不能起诉A公司（商标所有人）
- A公司不承担产品责任

**法律依据**：
- 产品责任由生产者承担
- 商标授权不导致责任转移

### 案例类型二：自产自销

**案情**：
A公司既是某品牌的商标注册人，又是自己生产产品的制造商。产品存在缺陷导致消费者B受伤。

**裁判结果**：
- B可以起诉A公司
- A公司作为生产者应承担产品责任

**法律依据**：
- 商标所有人与生产者为同一主体
- A公司应承担产品责任

### 案例类型三：挂牌生产争议

**案情**：
A公司委托B公司加工生产A公司品牌的产品，C购买后发现产品缺陷。A公司认为其只是商标授权方，不是生产者。

**裁判要点**：
- 需要审查A公司是否对生产进行实质控制
- 如果A公司仅提供商标标识而未参与生产，不承担责任
- 如果A公司对原材料、生产工艺等有决定权，可能承担责任
