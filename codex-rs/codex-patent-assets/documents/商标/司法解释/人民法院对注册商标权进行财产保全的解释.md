# 最高人民法院关于人民法院对注册商标权进行财产保全的解释

## 立法背景与目的

### 制定背景

在民事诉讼和执行程序中，人民法院需要对商标权进行财产保全时，需要明确：
- 保全的范围和方式
- 协助执行机关的义务
- 当事人的权利救济

### 规范目的

| 目的 | 说明 |
|------|------|
| 明确保全范围 | 确定哪些商标权可以保全 |
| 规范保全程序 | 确保保全措施依法进行 |
| 保护当事人权益 | 为当事人提供救济途径 |
| 协调执行机关 | 明确商标局协助执行义务 |

## 条款详解

### 第1条　保全范围

> 人民法院对注册商标权进行财产保全，应当包括：

**可以保全的商标权**：

| 类型 | 说明 |
|------|------|
| 注册商标专用权 | 商标注册人享有的专有使用权 |
| 商标许可使用权 | 许可使用合同项下的权利 |
| 商标转让权 | 转让注册商标的权利 |
| 商标质权 | 商标专用权质押 |

**保全客体**：

```
保全客体 = 商标权本体 + 商标权衍生物
```

| 衍生物 | 说明 |
|--------|------|
| 许可使用费 | 许可合同约定的使用费 |
| 转让费 | 转让合同约定的转让费 |
| 侵权赔偿金 | 因侵权行为获得的赔偿 |
| 商标价值 | 商标本身的价值 |

> [!important] 保全对象
> 保全对象是被执行人（商标注册人）名下的注册商标，以及与该商标相关的财产权益。

---

### 第2条　保全程序

> 人民法院对注册商标权采取财产保全措施，应当遵循民事诉讼法及其司法解释的有关规定。

**保全程序**：

| 阶段 | 内容 | 依据 |
|------|------|------|
| 申请 | 当事人申请或法院依职权 | 民事诉讼法第100条、第101条 |
| 审查 | 审查保全必要性 | 担保审查、紧急性审查 |
| 裁定 | 作出保全裁定 | 裁定书 |
| 执行 | 实施保全措施 | 协助执行通知书 |

**申请条件**：

| 条件 | 说明 |
|------|------|
| 紧急性 | 不立即采取保全措施会使合法权益受到难以弥补的损害 |
| 胜诉可能性 | 案件具有胜诉的可能 |
| 担保 | 申请人提供担保（法院要求时） |

> [!tip] 诉前保全
> 利害关系人在起诉前申请财产保全的，应当提供担保，不提供担保的，驳回申请。

---

### 第3条　保全裁定

> 人民法院裁定采取财产保全措施，应当同时作出裁定，裁定书应当送达双方当事人。

**裁定内容**：

| 内容 | 说明 |
|------|------|
| 当事人信息 | 申请人、被申请人 |
| 保全标的 | 具体商标（注册号、名称） |
| 保全措施 | 冻结、查封等具体措施 |
| 保全期限 | 保全期间 |
| 救济途径 | 复议/异议权利告知 |

**送达**：

| 对象 | 送达方式 |
|------|----------|
| 申请人 | 直接送达 |
| 被申请人 | 直接送达 |
| 商标局 | 协助执行通知书 |

---

### 第4条　保全措施

> 采取财产保全措施，可以采用冻结、查封或者法律允许的其他方法。

**具体措施**：

| 措施 | 适用情形 | 效果 |
|------|----------|------|
| 冻结 | 银行账户、应收债权 | 禁止转移、支付 |
| 查封 | 实物资产 | 禁止转移、处置 |
| 扣押 | 动产 | 扣留并控制 |
| 禁止转让 | 商标权 | 禁止转让、许可 |

**商标权保全的特殊措施**：

| 措施 | 说明 |
|------|------|
| 冻结商标转让 | 禁止办理转让登记 |
| 冻结许可备案 | 禁止办理许可备案 |
| 冻结质权登记 | 禁止办理质押登记 |
| 禁止使用 | 禁止在商业活动中使用 |

> [!warning] 保全期间
> 财产保全的期限：
> - 银行存款：不超过1年
> - 不动产：不超过3年
> - 动产：不超过2年
> - 商标权：参照不动产期限

---

### 第5条　续保与解除

> 财产保全期限届满，人民法院需要继续采取保全措施的，应当在期限届满前向当事人送达续保通知书。

**续保**：

| 要点 | 说明 |
|------|------|
| 申请时间 | 保全期限届满前 |
| 审查 | 审查续保必要性 |
| 裁定 | 作出续保裁定 |
| 执行 | 送达续保裁定 |

**解除保全**：

| 情形 | 依据 |
|------|------|
| 申请人撤诉 | 当事人申请 |
| 判决驳回诉讼请求 | 生效判决 |
| 保全错误 | 法院依申请或依职权 |
| 和解结案 | 双方和解 |
| 保全期限届满未续保 | 期限届满 |

> [!tip] 超期后果
> 保全期限届满未续保的，保全措施自动失效。

---

### 第6条　协助执行

> 商标局收到人民法院的协助执行通知书后，应当协助执行。

**协助执行内容**：

| 协助事项 | 说明 |
|----------|------|
| 冻结转让 | 停止办理转让手续 |
| 冻结许可 | 停止办理许可备案 |
| 冻结质权 | 停止办理质押手续 |
| 信息查询 | 提供商标权利状态 |

**送达方式**：

| 方式 | 说明 |
|------|------|
| 法院直接送达 | 执行法院送达 |
| 邮寄送达 | 通过 EMS 邮寄 |
| 电子送达 | 通过执行网络平台 |

> [!important] 商标局义务
> 商标局收到人民法院的协助执行通知书后，应当立即协助执行，不得拖延或者推诿。

---

### 第7条　执行异议

> 当事人对财产保全裁定不服的，可以申请复议一次。

**异议类型**：

| 类型 | 期限 | 审查法院 |
|------|------|----------|
| 复议 | 裁定送达之日起5日内 | 作出裁定的法院 |
| 异议 | 执行过程中 | 执行法院 |
| 执行异议之诉 | 异议裁定送达之日起15日内 | 执行法院 |

**异议理由**：

| 理由 | 说明 |
|------|------|
| 超范围保全 | 保全超出判决确定的范围 |
| 错误保全 | 保全对象错误 |
| 程序违法 | 保全程序违法 |
| 超额保全 | 保全价值超出诉讼标的 |

---

### 第8条　法律责任

> 人民法院工作人员违法采取保全措施，给当事人造成损失的，应当依法承担赔偿责任。

**责任类型**：

| 类型 | 适用情形 |
|------|----------|
| 国家赔偿 | 违法采取保全措施造成损害 |
| 民事责任 | 故意或重大过失造成损害 |
| 行政责任 | 构成犯罪的，追究刑事责任 |

**赔偿范围**：

| 项目 | 说明 |
|------|------|
| 直接损失 | 保全标的的直接损失 |
| 间接损失 | 因保全造成的其他损失 |
| 利息损失 | 资金被冻结期间的利息 |

> [!warning] 赔偿程序
> 当事人申请国家赔偿的，应当按照国家赔偿程序进行。

## 📊 实务要点汇总

### 申请财产保全流程

| 步骤 | 内容 | 注意事项 |
|------|------|----------|
| 1 | 准备申请材料 | 保全申请书、证据材料、担保 |
| 2 | 向法院提交申请 | 向有管辖权的法院提交 |
| 3 | 法院审查 | 法院在48小时内作出裁定 |
| 4 | 裁定执行 | 法院执行保全措施 |
| 5 | 送达协助执行 | 法院向商标局送达协助执行通知书 |

### 担保要求

| 情形 | 担保要求 |
|------|----------|
| 诉前保全 | 必须提供担保 |
| 诉讼保全 | 根据法院要求提供担保 |
| 执行保全 | 根据法院要求提供担保 |

**担保形式**：
- 现金担保
- 财产担保
- 保险公司保函
- 银行保函

### 保全执行

| 要点 | 内容 |
|------|------|
| 执行法院 | 采取保全措施的法院 |
| 协助机关 | 商标局 |
| 执行期限 | 裁定送达后立即执行 |
| 续保申请 | 期满前提出 |
