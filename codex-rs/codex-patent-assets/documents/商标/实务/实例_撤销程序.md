# 商标审查实例 - 撤销程序（第381-422页）

> 本卡片整理商标撤销程序的审查标准、典型案例及实务要点，涉及第381-422页内容。

## 📋 撤销程序概述

### 撤销类型

| 撤销类型 | 依据 | 原因 |
|----------|------|------|
| 撤销注册 | 商标法第49条 | 自行改变注册事项 |
| 撤销注册 | 商标法第49条 | 成为通用名称 |
| 撤销注册 | 商标法第49条 | 无正当理由连续三年不使用 |

### 法律依据

**《商标法》第49条**：
> 商标注册人在使用注册商标的过程中，自行改变注册商标、注册人名义、地址或者其他注册事项的，由地方工商行政管理部门责令限期改正；期满不改正的，由商标局撤销其注册商标。

> 注册商标成为其核定使用的商品的通用名称或者没有正当理由连续三年不使用的，任何单位或者个人可以向商标局申请撤销该注册商标。商标局应当自收到申请之日起九个月内做出决定。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长三个月。

## 📊 撤销程序类型

### 一、撤销连续三年不使用

**适用条件**：
- 注册商标连续三年未使用
- 无正当理由

**程序**：
```
撤销申请提出（任何人）
        ↓
商标局受理
        ↓
送达注册人（提供使用证据通知书）
        ↓
注册人提交使用证据（2个月内）
        ↓
商标局审查
        ↓
作出决定
        ↓
（可选）撤销复审
```

**时限**：
- 审查时限：9个月（可延长3个月）
- 复审时限：收到决定之日起15日内

### 二、撤销成为通用名称

**适用条件**：
- 注册商标成为核定使用商品的通用名称

**程序**：
- 任何人可提出
- 商标局审查认定

### 三、撤销自行改变注册事项

**适用条件**：
- 自行改变注册商标
- 自行改变注册人名义/地址
- 自行改变其他注册事项

**程序**：
- 地方工商行政管理部门责令改正
- 期满不改正，商标局撤销

## 📝 撤销三年不使用审查

### 审查标准

**"使用"的认定**：

> 商标的使用，是指将商标用于商品、商品包装或者容器以及商品交易文书上，或者将商标用于广告宣传、展览以及其他商业活动中，用于识别商品来源的行为。

**使用形式**：

| 使用形式 | 说明 |
|----------|------|
| 自行使用 | 商标注册人本人使用 |
| 许可使用 | 许可他人使用（需有许可合同） |
| 真实使用意图 | 有实际使用准备，因客观原因未使用 |

**不使用正当理由**：

| 正当理由 | 说明 |
|----------|------|
| 不可抗力 | 自然灾害、战争等 |
| 政府政策性限制 | 产业政策调整等 |
| 破产清算 | 企业破产清算期间 |
| 其他不可归责事由 | 非注册人主观原因 |

### 证据要求

**使用证据类型**：

| 证据类型 | 证明内容 |
|----------|----------|
| 销售合同/发票 | 商品销售时间、金额 |
| 广告合同/发票 | 广告宣传时间、范围 |
| 产品/包装 | 实物证据 |
| 展会参展证据 | 展会宣传 |
| 许可合同 | 许可使用关系 |

**证据特征**：
- 体现商标标识
- 体现核定使用商品
- 有明确的时间
- 有实际的商业使用

## 🔍 典型撤销案例

### 案例类型一：三年未使用撤销

**案情**：
- 注册人：某公司
- 注册商标："某品牌"
- 注册时间：5年前
- 撤销申请人：个人

**撤销理由**：
- 连续三年未使用

**注册人答辩**：
- 提供销售合同、发票等使用证据

**审理结果**：
- 证据充分：维持注册
- 证据不足：撤销注册

---

### 案例类型二：成为通用名称

**案情**：
- 注册商标："优盘"
- 核定商品：计算机存储设备

**撤销理由**：
- "优盘"已成为计算机存储设备的通用名称

**审理结果**：
- 认定构成通用名称：撤销注册
- 不构成通用名称：维持注册

---

### 案例类型三：自行改变注册事项

**案情**：
- 注册人将注册商标文字稍作修改后使用
- 注册人名义变更未办理变更手续

**撤销理由**：
- 自行改变注册商标
- 未办理名义变更

**审理结果**：
- 责令改正
- 期满不改：撤销注册

## 📌 撤销答辩要点

### 答辩策略

| 答辩类型 | 策略要点 |
|----------|----------|
| 实际使用 | 提供真实的使用证据 |
| 正当理由 | 提供不可抗力、政策限制等证据 |
| 许可使用 | 提供许可合同及实际使用证据 |
| 使用意图 | 提供使用准备但未实际使用的证据 |

### 证据清单

| 证据类型 | 具体材料 | 要求 |
|----------|----------|------|
| 销售证据 | 合同、发票、出货单 | 体现商标、商品、时间 |
| 广告证据 | 广告合同、发票、截图 | 体现商标、广告时间 |
| 产品证据 | 实物照片、包装 | 体现商标、商品 |
| 许可证据 | 许可合同、备案证明 | 体现许可关系 |
| 证明材料 | 检测报告、参展文件 | 体现商业活动 |

## ⚖️ 撤销的法律后果

### 效力

> 被撤销的注册商标，由商标局予以公告，该注册商标专用权自公告之日起终止。

### 禁止注册

> 注册商标被撤销、被宣告无效或者期满不再续展的，自撤销、宣告无效或者注销之日起**一年内**，商标局对与该商标相同或者近似的商标注册申请，**不予核准**。
