## 📊 显著性判断标准汇总

### 立体商标

| 要点 | 说明 |
|------|------|
| 使用方式 | 需要说明作为产品/包装/容器使用 |
| 指定商品 | 明确核定使用的商品类别 |
| 形状要求 | 不能是商品功能所决定的功能性形状 |

### 声音商标

| 要点 | 说明 |
|------|------|
| 描述方式 | 五线谱/简谱/文字描述 |
| 技术参数 | 音频、拍号、节奏、时长 |
| 曲调说明 | 大小调、旋律走向 |
