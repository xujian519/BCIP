项目概述

BCIP (Brand Copyright & Intellectual Property) 是一个基于 AI 的专利智能体平台，集成了专利检索、技术分析、文稿起草和审查功能。

项目使用 Rust 构建后端核心，React + TypeScript 构建桌面端界面，基于 Tauri 实现跨平台桌面应用。

项目结构
├── apps/desktop/        # Tauri 桌面应用
│   ├── src/             # React 前端
│   └── src-tauri/       # Rust 后端
├── codex-rs/            # Rust 核心库
├── codex-patent-domain/ # 专利领域模型
├── codex-patent-knowledge/ # 知识库
└── docs/                # 设计文档

核心功能

平台通过多阶段工作流（检索 → 分析 → 起草 → 审查）完成专利全生命周期管理。每个阶段都有对应的 AI Agent 负责执行具体任务。

知识库内置了专利法规、审查指南和判例数据，支持实时检索和上下文增强生成（RAG）。
