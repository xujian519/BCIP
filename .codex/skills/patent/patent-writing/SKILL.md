# 专利撰写能力

发明理解→权利要求布局→说明书撰写（技术领域、背景技术、发明内容、附图说明、具体实施方式）。

确保权利要求清楚简要，说明书充分公开，实施例至少3个。

## 质量检查

完成后自检：1）权利要求是否清楚简要 2）说明书是否充分公开 3）实施例是否充分 4）附图引用是否正确 5）形式是否规范。

## 输出格式

分析报告应包含：标题、分析日期、分析对象、分析方法、分析步骤、结论、置信度标注。使用结构化格式，便于审查。
