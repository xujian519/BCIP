# 答复策略能力

分析审查意见→确定答复策略（争辩、修改、组合）→使用标准模板→形成答复文本。

策略选择需考虑检索结果、修改可能性、审查倾向。

## 人机协作协议

以下节点必须人工确认：1）检索式确认 2）对比文件筛选 3）驳回策略选择 4）权利要求修改 5）最终文件审核。

## 输出格式

分析报告应包含：标题、分析日期、分析对象、分析方法、分析步骤、结论、置信度标注。使用结构化格式，便于审查。
