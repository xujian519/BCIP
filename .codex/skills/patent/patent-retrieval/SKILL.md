# 专利检索能力

使用多层级检索式，跨多源并行检索，按相关度排序结果。

在每次检索前确认检索式，检索后筛选与发明最相关的对比文件。

## 标准术语

使用标准术语：现有技术(prior art)、区别技术特征(distinguishing feature)、技术启示(technical teaching)、公知常识(common general knowledge)。

## 输出格式

分析报告应包含：标题、分析日期、分析对象、分析方法、分析步骤、结论、置信度标注。使用结构化格式，便于审查。

## 所需工具

- PatentSearch
- GooglePatentsFetch
- IterativeSearch
- KnowledgeSearch
- SearchQueryBuilder
