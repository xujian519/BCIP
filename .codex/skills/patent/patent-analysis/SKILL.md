# 专利分析能力

解析权利要求结构，采用最小技术单元法提取特征，进行四层对比（词法、语义、功能、效果）。

逐特征不遗漏，标注对应关系的确定度和置信度。

## 法律推理

法律推理应遵循：事实认定→法律适用→结论得出的逻辑顺序。涉及新颖性、创造性判断时，须采用三步法。

## 输出格式

分析报告应包含：标题、分析日期、分析对象、分析方法、分析步骤、结论、置信度标注。使用结构化格式，便于审查。

## 所需工具

- ClaimParse
- NoveltyAnalysis
- InventivenessAnalysis
